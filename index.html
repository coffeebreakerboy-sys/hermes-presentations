Key Learnings Saved

What Gets Tracked
• NOT saved: task progress, PR numbers, temporary TODOs

• PROCEDURES → saved as skills, not memory
• Memory limit: ~1.375 KB for user prefs

Challenges Encountered

Problems solved and lessons learned

✓ Hindsight import error — Missing hindsight package. Built a JSON-file shim to replace torch/VC++ dependency. Provider now local-embedded ✓
✓ GitHub PAT scope — First token lacked repo scope. Created correct token via .env file. Repo & Pages now live ✓
✓ Hosting services down — transfer.sh, 0x0.st, tmpfiles.org, file.io all failed. Pivoted to GitHub Pages for permanent hosting ✓
✓ Discord gateway failures — DNS resolution issue on restart. Fixed via debugging skills ✓
✓ Shell token mangling — Special chars in PAT broke shell commands. Switched to Python urllib with direct token in variable ✓
⚠ Discord CDN expiry — Attachment URLs return 404 after time. Lesson: never use CDN links for permanent hosting
